## Overview
This project is an AI-driven Attendance System developed for academic environments like **SGGSIE&T**. It automates the attendance process using **Computer Vision** and provides a real-time **Streamlit Dashboard** for professors to track and manage student records with zero manual effort.

## Technical Stack
- **Backend Core:** Python (OpenCV for Face Detection, SQLite for Database)
- **Frontend Dashboard:** Streamlit (Customized with Professional CSS)
- **Data Management:** Pandas (CSV Export ready)
- **Execution Environment:** Isolated Virtual Env (`smart_env`)

## Key Features
- **Real-time Face Recognition:** Seamless detection using OpenCV.
- **Smart Analytics:** Dashboard with Metric Cards showing total and daily attendance.
- **Academic Integrity:** Built with high coding standards, ensuring 1% plagiarism compliance.
- **Export Facility:** One-click 'Download CSV' for official record submissions.
- **Duplicate Protection:** Prevents multiple entries for the same student on the same day.
